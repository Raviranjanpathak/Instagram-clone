A simple Instagram-like web application built using HTML, CSS, JavaScript, Node.js, Express, and REST APIs. This project demonstrates CRUD operations where users can create, edit, view, and delete posts.

🚀 Features

📝 Create a new post with username, content, and image

✏️ Edit existing posts

❌ Delete posts

❤️ Display likes and 💬 comments

📄 View post details

📷 Upload images

🔄 RESTful routing (CRUD operations)

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express.js

Template Engine: EJS

Architecture: REST API
